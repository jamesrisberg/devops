# devops

Development Environment Topology Visualizer - A TUI tool to visualize, analyze, and clean up your dev environment.

## Installation

Local:
```bash
pip install -e .
```

Homebrew:
```bash
brew install jamesrisberg/devops/devops
```

## Usage

```bash
devops
```

## Features

- Visualize PATH entries and their sources
- View shell config files  
- Detect issues (duplicates, broken symlinks, conflicts)
- Clean up with confirmation
- View, update and uninstall packages from homebrew, npm, and other major package managers
- Build and run complex ffmpeg and imagemagick commands from simple forms
